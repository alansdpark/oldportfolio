Plants Vs Robots

Java

Plants Vs Robots was my final project for AP Computer Science in 11th Grade. Using Java, I was able to create a 2D game using sprites I created. This game is based on a popular desktop and cellular game, Plants Vs Zombies, where waves of "zombies" come and try to enter your home. Check out the game on GitHub!

Sticky Tongue

C#

Sticky Tongue is one of my first games made using a game engine. I used Unity3D to create a 2D game using sprites my friend created, and the goal of the game is to catch flies as they pass by, and avoid bees. It features a high score system saved on your PC, and has a functional main menu/pause menu.
